# DaurApp

DaurApp adalah aplikasi bank sampah yang menerima sampah daur ulang menjadi DaurPoin.
DaurPoin ini bisa ditukarkan menjadi uang yang keuntungannya semakin besar berdasarkan loyalitas pengguna.
